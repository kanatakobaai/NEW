## アプリ概要

### コンセプト
「財布の番人」— 衝動買いを止め、自分の消費パターンを見せる

### コアループ
```
欲しいと思う → アプリに登録 → 封印される
→ 冷却期間終了通知「本当に必要か？」
→ 不要 = 勝利記録 / 購入 = 敗北記録
→ 統計：節約総額・衝動傾向の可視化
```

### 画面構成
1. ホーム：封印中リスト・残り時間
2. 登録：商品名・金額・カテゴリ・冷却期間
3. 判定：期間終了後の Yes/No
4. 記録：勝利/敗北の履歴
5. 統計：節約額・パターン分析

### AdMob配置
| 広告種別 | タイミング |
|---------|---------|
| バナー | ホーム・記録画面 |
| インタースティシャル | 「不要だった」判定後 |
| リワード | 統計の詳細分析を解放 |
