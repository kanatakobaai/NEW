# AI Company

AIによって運営される会社プロジェクト。

## 組織構造

- **オーナー / 取締役**: kanata
- **AI CEO**: Jobs

## ディレクトリ構成

```
.
├── governance/          # ガバナンス（定款、取締役会議事録）
├── organization/        # 組織体制
│   └── ceo/            # CEO Jobs のプロフィール・方針
├── operations/          # 経営オペレーション
│   ├── decisions/      # 意思決定の記録
│   └── reports/        # レポート・報告
├── communications/      # オーナーとCEO間のコミュニケーション記録
└── projects/            # 進行中のプロジェクト
```

## 運営方針

オーナー(kanata)が取締役会として方針を示し、AI CEO(Jobs)が経営判断・実行を行う。
